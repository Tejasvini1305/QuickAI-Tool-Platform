# 🚀 QuickAI – AI SaaS Platform

QuickAI is a full-stack AI-powered SaaS platform that helps users generate content, create images, and enhance productivity using advanced AI tools.

## 🌟 Features

* ✍️ **AI Article Generator** – Generate high-quality articles instantly
* 🧠 **Title Generator** – Create catchy titles for blogs & posts
* 🖼️ **Image Generation** – Generate images using AI prompts
* 🧹 **Background Remover** – Remove image backgrounds easily
* 🧽 **Object Removal Tool** – Remove unwanted objects from images
* 🛠️ **Resume Viewer Tool** - Views Resume and gives suggestions
* 🔐 **Authentication System** – Secure user login/signup
* 💳 **Subscription System** – Premium feature access
* 📊 **Dashboard** – Manage user activity and tools

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js

### Database

* Neon Postgres

### APIs / Services

* AI APIs (OpenAI / ClipDrop / others)
* Cloudinary (for image storage)

## 📁 Project Structure

/client → React frontend
/server → Node.js backend

## 🚀 Deployment

* Frontend: Vercel / Netlify
* Backend: Render / Railway
* Database: Neon Postgres
